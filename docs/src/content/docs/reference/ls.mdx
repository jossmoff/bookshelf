### Usage

```bash
bookshelf ls [OPTIONS]
```
### Description

Use `bookshelf ls` to list all the current stories on your bookshelf. You can also filter the stories by tags.

### Options

| Name              | Shorthand | Default | Description                                                                                                         |
|-------            | --------- | ------- | -----------                                                                                                         |
| `--with-tags`|   |  | Comma-separated list of tags that you wish to filter in. |
