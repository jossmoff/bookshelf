title: bookshelf rm

### Usage
```bash
bookshelf rm [OPTIONS] STORY_NAME
```
### Description

Use `bookshelf rm` to remove a story from your bookshelf. This will delete the story from your bookshelf and all of its data, so be careful when using this command.
