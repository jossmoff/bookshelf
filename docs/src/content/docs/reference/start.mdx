### Usage

```bash
bookshelf start [OPTIONS] STORY_NAME
```

### Description

Use `bookshelf start` to start a new chapter for a story on your bookshelf.
By starting, you are starting a live for the chapter and by proxy the story.
The live duration for the story will be shown to you as a result of the command.
