---

### Usage

```bash
bookshelf stop [OPTIONS] STORY_NAME
```

### Description
Use `bookshelf stop` to stop the current chapter of a story on your bookshelf. This will stop the timer and save the elapsed time to the story.

| Name              | Shorthand | Default | Description|
|-------            | --------- | ------- | -----------|
|   |   |  |  |
